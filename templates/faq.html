About this Project

aka infrequently asked questions

What does this website do?

It's software that generates random poetry (based on a source text) following some common poem formats! It does its best to make sure the poems it generates rhyme and follow the correct meter (stressed and unstressed syllables) for the format.

How does it work?

The source text is analyzed, and words for the poem are strung together randomly using a Markov chain strategy -- so there's an awareness of neighboring words and the frequency they occur in the original text. Rhyme and meter for each word are looked up using the CMU Pronunciation dictionary, which offers a "word-to-sounds" translation that software can interpret.

For more information, check out the PyCon 2020 talk!

There's also some custom code to guess the number of syllables in a word if there's a word that isn't found in the dictionary, which is mostly based on the number of vowel groups in the word.
